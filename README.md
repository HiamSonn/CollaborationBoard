# OpenCollaBoard

OpenCollaBoard is a collaborative web application, similar to Miro and Figma Jam, designed for creating flowcharts, mind maps, wireframes, and other visual diagrams in real time. With OpenCollaBoard, users can work together simultaneously, creating, editing, and collaborating on visual projects with drag-and-drop ease, adding notes, comments, images, and text to enhance visual communication.

---

### Technologies

OpenCollaBoard was developed using the following technologies:

- [Next.js](https://nextjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [React Flow](https://reactflow.dev/)
- [Socket.io](https://socket.io/)
- [Node.js](https://nodejs.org/)

---

### Installation and Usage

To install and run OpenCollaBoard, follow these steps:

1.  Clone the repository.
2.  Install dependencies using npm install inside both the frontend and server directories.
3.  Start the server using npm run dev inside the server directory. It will be available at http://localhost:4000.
4.  Start the frontend using npm run dev inside the frontend directory. It will be available at http://localhost:3000.
